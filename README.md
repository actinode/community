## How to Report an Issue

1. Check [existing issues](https://github.com/actinode/community/issues)
2. Check [documentation](https://docs.actinode.com)
3. [Open a new issue](https://github.com/actinode/community/issues/new/choose)

## Issue Types

- 🐛 Bug Report — something is broken
- 🚀 Feature Request — suggest a new feature
- 📦 Package Request — suggest a new package
